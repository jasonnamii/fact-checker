# 허용 임계 (Tolerance Guide)

⚠️ 미세오차 판정 시 문서 용도별 허용 범위.

---

## 용도별 허용 기준

| 용도 | 금액 | 비율 | 일자 | 규모 |
|------|------|------|------|------|
| **IR/공시** | 원 단위 정확 | 소수점 2자리 | 정확한 일자 | 정확 |
| **투자 제안서** | 백억 반올림 OK | 정수% OK | 월 단위 OK | 단위 정확 필수 |
| **내부 전략 문서** | 천억 OK | 5%p 이내 | 분기 OK | 대략적 OK |
| **대외 발표/PR** | 백억 반올림 OK | 정수% OK | 월 단위 OK | 단위 정확 필수 |

## 판정 규칙

| 조건 | 판정 |
|------|------|
| 허용 범위 이내 | ⚠️ 미세오차 — "정확한 수치는 X" 주석 권고 |
| 허용 범위 초과 | ❌ 수정필요 |
| 단위 자체가 다름 (평↔㎡, 매출↔거래액) | ❌ 수정필요 |

## 반올림 표기 권고
❌ "매출 1조 5,600억" (정확처럼 보이지만 실제 1조 5,578억)
✅ "매출 약 1조 5,600억" 또는 "매출 1조 5,578억"
